Welcome to Group Organizer

This is a webpage designed to help you organize people into groups

How it works:

  1. Enter the names of the people you would like to organize into groups
  2. Check the box for "Groups" if you would like to divide people into a selected number of groups (if you select 4, there will be a total of 4 groups)
  3. Check the box for "People" if you would like to assign a certain number of people to each group (if you select 3, there will be 3 people per group)
  4. Select the number of people or groups to divide by
  5. Hit "Submit"
